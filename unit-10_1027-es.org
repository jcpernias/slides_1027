* Selecciﾃｳn del nﾃｺmero de retardos


** Introducciﾃｳn

- Modelo ARDL($p$, $q$):
  #+begin_export latex
  \[
     y_t = \alpha + \rho_1 y_{t-1} + \dots + \rho_p y_{t-p} +
     \beta_0 z_t + \beta_1 z_{t-1} + \dots + \beta_q z_{t-q} +
     u_t
  \]
  #+end_export

- ﾂｿCuﾃ｡ntos retardos hay que incluir en una regresiﾃｳn de series
  temporales?

- ﾂｿQuﾃｩ mﾃｩtodos podemos usar para determinar el nﾃｺmero de retardos?


** Selecciﾃｳn de retardos

- Si se selecciona un nﾃｺmero demasiado pequeﾃｱo de retardos se
  omite informaciﾃｳn relevante.

- Con un nﾃｺmero demasiado grande se pierde eficiencia.


** Selecciﾃｳn de modelos

- Se establece un nﾃｺmero mﾃ｡ximo de retardos lo suficientemente alto
  como para capturar todas las relaciones dinﾃ｡micas.

- Se estiman modelos con distinto nﾃｺmero de retardos. *Todas las
  regresiones deben estimarse con la misma muestra*.

- Se utiliza un indicador para determinar cuﾃ｡l es el modelo mﾃ｡s
  adecuado.


** Contrastes F

- Se contrasta la significaciﾃｳn del ﾃｺltimo retardo usando un
  contrasteﾂ$F$. Si el ﾃｺltimo retardo no es significativo, se estima
  un nuevo modelo sin ese retardo y se vuelve a repetir el proceso.

- Los contrastes $F$ no estﾃ｡n diseﾃｱados para seleccionar modelos y el
  procedimiento tiende a seleccionar mﾃ｡s retardos de los necesarios.


** Criterios de informaciﾃｳn

- En la selecciﾃｳn de modelos se suele usar algﾃｺn *criterio de
  informaciﾃｳn*: medidas de la bondad del ajuste que incluyen una
  penalizaciﾃｳn creciente en el nﾃｺmero de parﾃ｡metros.

- Se estiman todos los modelos que se quieran comparar. Se selecciona
  aquel con un menor valor del criterio de informaciﾃｳn.


** Algunos criterios de informaciﾃｳn

- *AIC*: criterio de informaciﾃｳn de Akaike. En los modelos de
  regresiﾃｳn puede escribirse como:
  #+begin_export latex
  \[
     \text{AIC} = C + T \log\left(\frac{\SCR}{T}\right) + 2 (K + 1),
  \]
  #+end_export
  donde $C$ es una constante, $T$ es el nﾃｺmero de observaciones,
  $\SCR$ es la suma del cuadrado de los residuos y $K$ es el nﾃｺmero de
  parﾃ｡metros.

- *BIC*: criterio de informaciﾃｳn bayesiano.
  #+begin_export latex
  \[
     \text{BIC} = C + T \log\left(\frac{\SCR}{T}\right) + \log(T) (K + 1).
  \]
  #+end_export
  Se diferencia del AIC en que penaliza mﾃ｡s el nﾃｺmero de parﾃ｡metros.


** Comparaciﾃｳn entre criterios

- Usando el BIC se obtiene una estimaciﾃｳn consistente del nﾃｺmero de
  retardos.

- El AIC produce resultados similares aunque suele seleccionar un
  nﾃｺmero de retardos mayor que el BIC.



* Regresiﾃｳn espuria


** Regresiﾃｳn espuria en cortes transversales

- Dos variables, $y$ y $x$, parecen estar muy correlacionadas y la
  regresiﾃｳn de $y$ sobre $x$ produce resultados significativos.

- La relaciﾃｳn entre $y$ y $x$ desaparece cuando se incluye una tercera
  variable, $z$, en la regresiﾃｳn.

- La regresiﾃｳn de $y$ sobre $x$ es un ejemplo de *regresiﾃｳn espuria*.


** Regresiﾃｳn espuria en series temporales

- La omisiﾃｳn de variables relevantes tambiﾃｩn puede producir
  regresiones espurias con datos de series temporales.

- Ademﾃ｡s, es muy frecuente encontrar regresiones espurias cuando se
  incluyen series temporales con tendencia o que sean altamente
  persistentes.


** Tipos de tendencia

La *tendencia* de una serie temporal estﾃ｡ constituida por movimientos
persistentes a largo plazo. Distinguimos entre:

- Tendencias deterministas.

- Tendencias estocﾃ｡sticas.


** Tendencias deterministas

- Una *tendencia determinista* es una funciﾃｳn no aleatoria del tiempo.

- Por ejemplo, si $y_t$ fluctﾃｺa alrededor de una *tendencia lineal*:
  #+begin_export latex
  \[
    y_t = \alpha + \delta t + u_t,
  \]
  #+end_export
  donde $u_t$ es un proceso estacionario y dﾃｩbilmente dependiente.


** Regresiﾃｳn y tendencias lineales

- Regresiﾃｳn con dos variables, $y_t$ y $x_t$, las cuales no estﾃ｡n
  relacionadas pero presentan *tendencias lineales*:
  #+begin_export latex
  \[
    y_t = \alpha + \beta x_t + u_t.
  \]
  #+end_export

- A pesar de que $\beta = 0$, la regresiﾃｳn anterior con frecuencia
  indicarﾃｭa que existe una relaciﾃｳn significativa entre las dos
  variables.

- Para evitar encontrar regresiones espurias, es aconsejable
  introducir una tendencia lineal en la regresiﾃｳn anterior:
  #+begin_export latex
  \[
    y_t = \alpha + \beta x_t + \delta t + u_t.
  \]
  #+end_export


** Tendencias estocﾃ｡sticas

- Una *tendencia estocﾃ｡stica* es aleatoria y varﾃｭa con el paso del
  tiempo.

- El ejemplo mﾃ｡s simple de una variable con tendencia estocﾃ｡stica es
  el *paseo aleatorio*:
  #+begin_export latex
  \[
    y_t = y_{t-1} + e_t.
  \]
  #+end_export

- Un paseo aleatorio puede alterna periodos prolongados de crecimiento
  y de decrecimiento.


** Paseo aleatorio con deriva

- Algunas series tienen una tendencia de crecimiento sostenido a largo
  plazo.

- *Paseo aleatorio con deriva*:
  #+begin_export latex
  \[
    y_t = \delta + y_{t-1} + e_t.
  \]
  #+end_export

- Si el parﾃ｡metro $\delta > 0$ la variable $y_t$ tiende a crecer con
  el paso del tiempo.


** Raﾃｭces unitarias

- Un paseo aleatorio es un caso especial de un \AR{1}, donde $\rho =
  1$, incumpliendo la condiciﾃｳn de estabilidad.

- La condiciﾃｳn de estabilidad de procesos \AR{p} es mﾃ｡s complicada y
  depende del valor de las raﾃｭces del *polinomio caracterﾃｭstico*: un
  polinomio de orden $p$ formado a partir de los coeficientes del
  proceso \AR{p}.

- Cuando una de esas raﾃｭces sea igual a 1, se dice que el proceso tiene
  una *raﾃｭz unitaria*.


** Raﾃｭces unitarias, tendencias estocﾃ｡sticas y procesos I(1)

- Si un proceso tiene una raﾃｭz unitaria entonces tiene una
  tendencia estocﾃ｡stica.

- La diferencia de un proceso con una raﾃｭz unitaria no tiene tendencia
  estocﾃ｡stica.

- Un proceso con una raﾃｭz unitaria es un proceso integrado de orden 1,
  I(1).


** Regresiﾃｳn espuria y tendencias estocﾃ｡sticas

- Si $y_t$ y $x_t$ son procesos I(1), la regresiﾃｳn de $y_t$ sobre
  $x_t$ mostrarﾃ｡ con mucha frecuencia resultados muy significativos
  aunque, en realidad, las dos variables no estﾃｩn relacionadas.

- Cuando $y_t$ y $x_t$ estﾃ｡n relacionadas y tienen una tendencia
  comﾃｺn, los residuos de la regresiﾃｳn de $y_t$ sobre $x_t$ deberﾃｭan
  comportarse como un proceso I(0). En ese caso las variables estﾃ｡n
  *cointegradas*.


** Regresiﾃｳn con variables con tendencias

- Se debe aﾃｱadir un tﾃｩrmino de tendencia lineal cuando alguna de las
  variables presente tendencias deterministas.

- Las variables con raﾃｭz unitaria deben ser diferenciadas para
  eliminar las tendencias estocﾃ｡sticas.

- La posibilidad de que haya relaciones de cointegraciﾃｳn entre
  variables I(1) no se abordarﾃ｡ en este curso.


* Contrastes de raﾃｭz unitaria


** Modelo AR(1)

El modelo autorregresivo de orden 1, \AR{1}, puede escribirse como:
#+begin_export latex
\[
  y_t = \alpha + \rho y_{t-1} + u_t,
\]
#+end_export
donde $u_t$ es un ruido blanco.


** Estacionariedad y dependencia dﾃｩbil

Las propiedades dinﾃ｡micas del proceso \AR{1} dependen del valor de $\rho$:

- $y_t$ es estacionario y dﾃｩbilmente dependiente si $\abs{\rho} < 1$.

- $y_t$ es un paseo aleatorio si $\rho = 1$.

- $y_t$ tiene un comportamiento explosivo cuando $\rho > 1$.


** Detecciﾃｳn de raﾃｭces unitarias

- Los estadﾃｭsticos habituales no pueden usarse para contrastar la
  existencia de una raﾃｭz unitaria.

- La estimaciﾃｳn MCO del parﾃ｡metro $\rho$ estﾃ｡ sesgada hacia abajo y el
  sesgo aumenta cuanto mﾃ｡s cerca estﾃ｡ $\rho$ de 1.

- La distribuciﾃｳn del estadﾃｭstico $t = (\hat{\rho} - 1)/\se({\rho})$
  es muy diferente de una $t_{T-2}$ o de una normal cuando $\rho$
  tiene un valor cercano a 1.



** Regresiﾃｳn de Dickey-Fuller

- Proceso AR(1):
  #+begin_export latex
  \[
    y_t = \alpha + \rho y_{t-1} + u_t,
  \]
  #+end_export


- Restando $y_{t-1}$ de ambos lados:
  #+begin_export latex
  \[
    \incr y_t = \alpha + (\rho - 1) y_{t-1} + u_t.
  \]
  #+end_export

- Dickey y Fuller plantean contrastar la existencia de una raﾃｭz
  unitaria estimado por MCO el modelo de regresiﾃｳn:
  #+begin_export latex
  \[
    \incr y_t = \alpha + \beta y_{t-1} + u_t.
  \]
  #+end_export


** Contraste $\bm{t}$ de Dickey-Fuller

- Regresiﾃｳn de Dickey y Fuller:
  #+begin_export latex
  \[
    \incr y_t = \alpha + \beta y_{t-1} + u_t.
  \]
  #+end_export

- La existencia de una raﾃｭz unitaria implica que $\beta = 0$.

- Dickey y Fuller proponen contrastar la hipﾃｳtesis nula $H_0\!: \beta =
  0$ frente a la  alternativa $H_1\!: \beta < 0$ mediante un
  contraste $t$ de una cola.


** Distribuciﾃｳn del contraste de Dickey-Fuller

- *No debe usarse un contraste $\bm{t}$ robusto a heteroscedasticidad*. Bajo
  la hipﾃｳtesis nula el contraste $t$ tradicional es vﾃ｡lido aunque haya
  heteroscedasticidad.

- El contraste de Dickey-Fuller bajo la hipﾃｳtesis nula no sigue una
  distribuciﾃｳn normal ni ninguna otra distribuciﾃｳn convencional.

- Los valores crﾃｭticos del contraste se obtienen mediante simulaciﾃｳn:
  #+ATTR_LATEX:  :align *{3}{@{\hspace{3mm}}r} :booktabs t
  | $10\%$  | $5\%$   | $1\%$   |
  |---------+---------+---------|
  | $-2.57$ | $-2.86$ | $-3.43$ |


** Contraste de Dickey-Fuller aumentado

- Si $y_t$ sigue un proceso \AR{p} es necesario incluir $p-1$ retardos
  de $\incr y_t$ en la regresiﾃｳn de Dickey-Fuller para obtener un
  contraste vﾃ｡lido:
  #+begin_export latex
  \[
    \incr y_t =
    \alpha + \beta y_{t-1} + \phi_1 \incr y_{t-1}  +
    \phi_2 \incr y_{t-2} + \dots  + \phi_{p-1} \incr y_{t-p+1}  + u_t.
  \]
  #+end_export

- Los valores crﾃｭticos de Dickey-Fuller son vﾃ｡lidos para contrastar
  $H_0\!: \beta = 0$ frente a $H_1\!: \beta < 0$.


** Selecciﾃｳn de retardos

- Es conveniente utilizar un criterio de informaciﾃｳn para seleccionar
  el nﾃｺmero de retardos a incluir en la regresiﾃｳn de Dickey-Fuller.

- Para el contraste de Dickey-Fuller aumentado es preferible el AIC.


** Tendencias

- Si $y_t$ presenta una tendencia creciente o decreciente es
  aconsejable aﾃｱadir una tendencia lineal a la regresiﾃｳn de
  Dickey-Fuller:
  #+begin_export latex
  \[
    \incr y_t =
    \alpha + \delta t + \beta y_{t-1} + \phi_1 \incr y_{t-1}  +
    \phi_2 \incr y_{t-2} + \dots  + \phi_{p-1} \incr y_{t-p+1}  + u_t.
  \]
  #+end_export

- Cuando se incluye una tendencia, la distribuciﾃｳn del contraste de
  Dickey-Fuller es diferente y es necesario usar otros valores
  crﾃｭticos:
  #+ATTR_LATEX:  :align *{3}{@{\hspace{3mm}}r} :booktabs t
  | $10\%$  | $5\%$   | $1\%$   |
  |---------+---------+---------|
  | $-3.12$ | $-3.41$ | $-3.96$ |



** Resumen

- Mediante la inspecciﾃｳn de grﾃ｡ficos, se determina si es necesario
  incluir una tendencia lineal en la regresiﾃｳn de Dickey-Fuller.

- Se selecciona el nﾃｺmero de retardos en la regresiﾃｳn de Dickey-Fuller
  de acuerdo con el valor del AIC.

- Se rechaza la presencia de raﾃｭz unitaria cuando el valor del
  cociente $t$ para $H_0\!: \beta = 0$ frente a $H_1\!: \beta < 0$ es
  menor que los valores crﾃｭticos de la siguiente tabla:
  #+ATTR_LATEX:  :align l*{3}{@{\hspace{3mm}}r} :booktabs t
  | Tendencia lineal | $10\%$  | $5\%$   | $1\%$   |
  |------------------+---------+---------+---------|
  | No               | $-2.57$ | $-2.86$ | $-3.43$ |
  | Sﾃｭ               | $-3.12$ | $-3.41$ | $-3.96$ |

** Multiplicadores de largo plazo (I)

- Modelo con tres retardos de la variable explicativa:
  #+begin_export latex
  \[
     y_t = \alpha + \beta_0 z_t + \beta_1 z_{t-1} + \beta_2 z_{t-2} + \beta_3 z_{t-3} + u_t.
  \]
  #+end_export

- Es posible reescribir el modelo anterior como:
  #+begin_export latex
  \[
     y_t = \alpha + \delta_0 \incr z_t + \delta_1 \incr z_{t-1} +
     \delta_2 \incr z_{t-2} + \delta_3 z_{t-3} + u_t
  \]
  #+end_export

- Con la regresiﾃｳn anterior se estiman directamente los
  multiplicadores acumulativos y de largo plazo y sus correspondientes
  errores tﾃｭpicos.


** Multiplicadores de largo plazo (y II)

- Modelo ARDL($p$, $q$):
  #+begin_export latex
  \[
     y_t = \alpha + \rho_1 y_{t-1} + \dots + \rho_p y_{t-p} +
     \beta_0 z_t + \beta_1 z_{t-1} + \dots + \beta_q z_{t-q} +
     u_t
  \]
  #+end_export

- El multiplicador de largo plazo es una funciﾃｳn no lineal de los
  parﾃ｡metros:
  #+begin_export latex
  \[
     \delta_{\text{LP}} = \frac{\sum_{j=0}^q \beta_j}{1 - \sum_{j=1}^p \rho_j}.
  \]
  #+end_export

- En este curso no se tratarﾃ｡ como obtener los multiplicadores
  acumulativos y sus errores tﾃｭpicos en modelos ARDL.


** Cambio estructural

- Se produce un *cambio estructural* cuando los parﾃ｡metros del modelo
  de regresiﾃｳn no permanecen constantes a lo largo de todo el periodo
  muestral.

- Consideraremos el caso de que los parﾃ｡metros sﾃｳlo cambian una vez y
  que conocemos en quﾃｩ periodo se produce el cambio.


** Contraste de Chow (I)

- Modelo de regresiﾃｳn original:
  #+begin_export latex
  \[
    y_t = \beta_0 + \beta_1 x_{1t}  + \beta_2 x_{2t} + u_t.
  \]
  #+end_export

- El *contraste de Chow* puede usarse cuando se sospecha de la
  existencia de un cambio estructural en el periodo $\tau$, $1 < \tau
  < T$.

- Creamos una variable ficticia $D_t$ que toma el valor 0 en todos
  los periodos anteriores a $\tau$ y valor 1 a partir del periodo
  $\tau$.
  #+begin_export latex
  \[
    D_t =
    \begin{cases}
      0 & \text{si $t < \tau$}, \\
      1 & \text{si $t \geq \tau$}.
    \end{cases}
  \]
  #+end_export


** Contraste de Chow (y II)

- Aﾃｱadimos al modelo original la variable $D_t$ asﾃｭ como
  interacciones de $D_t$ con todas las variables explicativas:
  #+begin_export latex
  \[
    y_t = \beta_0 + \beta_1 x_{1t}  + \beta_2 x_{2t} +
    \theta_0 D_t + \theta_1 D_t x_{1t}   + \theta_2 D_t x_{2t}
    + u_t.
  \]
  #+end_export

- Bajo la hipﾃｳtesis nula de que no se ha producido cambio estructural,
  los tﾃｩrminos que se han aﾃｱadido no son significativos.

- Puede usarse un estadﾃｭstico $F$ para contrastar $\theta_0 = \theta_1
  = \theta_2 = 0$.

- En caso que se rechace la hipﾃｳtesis nula, el parﾃ｡metro $\theta_j$
  muestra como ha cambiado el parﾃ｡metro $\beta_j$ a partir del periodo
  $\tau$.


** Estacionalidad

- Con datos trimestrales y mensuales es usual observar variables con
  fuertes oscilaciones estacionales.

- Una forma simple de controlar este tipo de variaciﾃｳn es incluir
  *variables ficticias estacionales* en el modelo de regresiﾃｳn.


** Variables ficticias estacionales

- Con datos trimestrales podemos definir cuatro variables ficticias
  estacionales $D^s_t$, $s = 1, \dots, 4$:
  #+begin_export latex
  \[
    D^s_t =
    \begin{cases}
      1 & \text{si $t$ corresponde al trimestre $s$}, \\
      0 & \text{en caso contrario}.
    \end{cases}
  \]
  #+end_export

- En la regresiﾃｳn sﾃｳlo incluirﾃｭamos tres de estas variables. De otra
  forma, incurrirﾃｭamos en la *trampa de las variables ficticias*.

- De la misma manera, para controlar la estacionalidad en datos
  mensuales incluirﾃｭamos un conjunto de 11 variables ficticias
  estacionales.


** COMMENT Desestacionalizaciﾃｳn

- Incluir variables ficticias en la regresiﾃｳn es equivalente a estimar
  el modelo con las variables previamente *desestacionalizadas*.

- Es frecuente encontrar variables que presentan oscilaciones
  estacionales y comportamiento tendencial. Ambos tipos de efectos
  pueden controlarse incluyendo simultﾃ｡neamente una tendencia lineal y
  variables ficticias estacionales.


** COMMENT Tendencias lineales

Incluir una tendencia lineal es equivalente a:

- Eliminar la tendencia lineal de la variable dependiente, calculando
  $\ddot{y}_t$, los residuos de la regresiﾃｳn:
  #+begin_export latex
  \[
    y_{t} = \ahat_{0} + \ahat_{1} t + \ddot{y}_{t}
  \]
  #+end_export

- De forma similar, eliminar la tendencia lineal de cada una de las
  variables explicativas obteniendo $\ddot{x}_{1t}, \ddot{x}_{2t}, \dots, \ddot{x}_{kt}$.

- Estimar por MCO los parﾃ｡metros de una regresiﾃｳn de $\ddot{y}_t$
  sobre $\ddot{x}_{1t}, \ddot{x}_{2t}, \dots, \ddot{x}_{kt}$.
